# zca-desktop v0.1.4

Release date: 2026-06-06

## Install

Download the universal macOS DMG from the GitHub release:

```sh
open Zalo.Desktop_0.1.4_universal.dmg
```

Or install through Homebrew Cask:

```sh
brew tap tuanle96/tap
brew upgrade --cask zca-desktop
```

## Highlights

- Fixed the immediate startup crash in v0.1.3.
- The magic-link callback listener now starts on Tauri's async runtime instead of calling `tokio::spawn` before a Tokio reactor exists.
