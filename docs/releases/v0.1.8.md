# zca-desktop v0.1.8

Release date: 2026-06-09

## Install

Download the universal macOS DMG from the GitHub release and open it:

```sh
open Zalo.Desktop_0.1.8_universal.dmg
```

Or install through Homebrew Cask:

```sh
brew tap tuanle96/tap
brew update --force --quiet
brew reinstall --cask tuanle96/tap/zca-desktop
```

The DMG is Developer ID signed, notarized, and stapled, so it opens normally on
macOS without a Gatekeeper workaround.

## Highlights

- The OAuth login screen now uses the real Google and GitHub brand glyphs
  (multi-color Google "G", GitHub mark) instead of generic placeholder icons,
  via a shared brand-mark component.

This is a UI-only follow-up to the v0.1.7 cloud OAuth device login; the backend
contract and login flow are unchanged.
