# zca-desktop v0.1.3

Release date: 2026-06-06

## Install

Download the universal macOS DMG from the GitHub release:

```sh
open Zalo.Desktop_0.1.3_universal.dmg
```

Or install through Homebrew Cask:

```sh
brew tap tuanle96/tap
brew install --cask zca-desktop
```

## Highlights

- Magic-link emails now use a browser landing page first, then hand the token to the running desktop app through the local callback listener.
- The desktop app validates the callback origin against the configured cloud API before accepting a magic-link token.
- Cloud server defaults now resolve to `https://zca.tuanle.dev`.
- Persisted localhost cloud server values are normalized back to the production cloud endpoint.
- Advanced cloud settings no longer keep loopback hosts as the effective cloud server.
