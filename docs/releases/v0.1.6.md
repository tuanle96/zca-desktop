# zca-desktop v0.1.6

Release date: 2026-06-06

## Install

Download the universal macOS DMG from the GitHub release:

```sh
open Zalo.Desktop_0.1.6_universal.dmg
```

Or install through Homebrew Cask:

```sh
brew tap tuanle96/tap
brew update --force --quiet
brew reinstall --cask tuanle96/tap/zca-desktop
```

## Highlights

- Fixed duplicate magic-link delivery from the browser landing page.
- Desktop now ignores duplicate callbacks for a magic link that was already verified successfully.
- The login gate now advances to the Zalo QR step after device-link verification, even when no Zalo account has been added yet.
