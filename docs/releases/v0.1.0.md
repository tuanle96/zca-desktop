# zca-desktop v0.1.0

Initial public desktop release for personal, noncommercial use.

## Download

- `Zalo Desktop_0.1.0_universal.dmg`
- SHA-256: `6ba5f4de1df45c8471154efd6e173c4525b53975136de4e021b264231321cf88`
- macOS architecture: universal `x86_64 arm64`
- Bundle identifier: `app.zca.desktop`

## Highlights

- Tauri v2 + Rust + SvelteKit desktop client for local Zalo chat.
- QR login with encrypted credential persistence in the OS keychain-backed local store.
- Multi-account session management with realtime listeners, restore, logout, and reauth-needed state.
- Local SQLite history for accounts, threads, messages, rich message metadata, attachment metadata, and recent stickers.
- Rich chat UI support for text, quoted replies, stickers, reactions, link previews, and recalled/deleted state.
- Contact and group metadata loading with thread identity backfill.
- Optional self-hosted cloud backend for magic-link device auth, hosted Zalo sessions, encrypted sync/media state, and SSE realtime.
- macOS DMG is Developer ID signed, notarized, stapled, and Gatekeeper accepted.

## Caveats

- Unofficial client, not affiliated with Zalo or VNG.
- Zalo credentials are bearer tokens; do not share imported credentials or debug captures.
- Using unofficial clients can create account-ban or account-limitation risk.
- Local desktop attachment sending remains on the roadmap.
- Windows and Linux signed installers are not included in this release.
