# zca-desktop v0.1.2

Release date: 2026-06-06

## Install

Download the notarized universal macOS DMG from the GitHub release:

```sh
open Zalo.Desktop_0.1.2_universal.dmg
```

Or install through Homebrew Cask:

```sh
brew tap tuanle96/tap
brew install --cask zca-desktop
```

## In-app updater

This release enables the Tauri v2 updater flow in the desktop app:

- Settings -> Giới thiệu checks `https://github.com/tuanle96/zca-desktop/releases/latest/download/latest.json`.
- The macOS updater payload is `Zalo.Desktop.app.tar.gz`.
- The updater payload is signed with the Tauri updater key and verified by the app before install.
- After install, the app relaunches through `@tauri-apps/plugin-process`.

## Highlights

- Added updater and process plugin registration in the Tauri shell.
- Added updater permissions to the default Tauri capability.
- Added Settings About UI for checking, downloading, installing, and relaunching updates.
- Published the signed updater manifest source at `docs/releases/latest.json`.

## Verification

- `bun run --cwd apps/desktop check`
- `cargo check --manifest-path apps/desktop/src-tauri/Cargo.toml`
- `bun --cwd apps/desktop tauri build --target universal-apple-darwin --bundles app,dmg --ci`
- `xcrun notarytool submit ... --wait` returned `Accepted`.
- `xcrun stapler validate` passed for the DMG.
- `codesign --verify` passed for the DMG and mounted app.
- `spctl` accepted the DMG and mounted app as `Notarized Developer ID`.
