# zca-desktop v0.1.5

Release date: 2026-06-06

## Install

Download the universal macOS DMG from the GitHub release:

```sh
open Zalo.Desktop_0.1.5_universal.dmg
```

Or install through Homebrew Cask:

```sh
brew tap tuanle96/tap
brew update --force --quiet
brew reinstall --cask tuanle96/tap/zca-desktop
```

## Highlights

- Fixed the cloud login screen still using `http://127.0.0.1:37880` on production installs.
- Production-safe cloud defaults now ignore loopback `PUBLIC_ZCA_CLOUD_BASE_URL` values.
- Stale `zca.cloud.baseUrl` browser storage values are migrated back to `https://zca.tuanle.dev` on app load.
