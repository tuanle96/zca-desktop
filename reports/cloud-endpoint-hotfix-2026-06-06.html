Zalo Desktop production hotfix

Cloud endpoint hotfix v0.1.5

Ngày phát hành: 06.06.2026. Bản này xử lý lỗi màn hình đăng nhập cloud vẫn dùng http://127.0.0.1:37880 trên production install.

Kết luận

Đã sửa lỗi localhost fallback và build lại artifact production v0.1.5.

Nguyên nhân là PUBLIC_ZCA_CLOUD_BASE_URL có thể được xuất vào dynamic env với giá trị localhost, và code cũ cho phép giá trị này trở thành default. Bản mới reject loopback default và migrate stale localStorage sang https://zca.tuanle.dev.

Thay đổi chính

  • Desktop cloud config không dùng loopback host làm default production.
  • Stale key zca.cloud.baseUrl được ghi đè lại bằng cloud endpoint hợp lệ khi app load.
  • Mobile cloud config áp dụng cùng production-safe resolver để tránh drift.
  • Version, changelog, release note, updater metadata và cask metadata đã nâng lên 0.1.5.
